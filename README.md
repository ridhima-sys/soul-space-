# 🌸 Soul Space – Mental Wellness Web App

A calming and interactive web application designed to support mental well-being.  
Soul Space provides a safe digital environment where users can express emotions, relax, and feel supported.

---

## ✨ Features

- 💬 Mood Selection (Sad, Anxious, Angry, Lonely, Hopeless)
- 📊 Dashboard with mood tracking history
- 🤝 Connect with volunteers based on mood
- 🎧 Relax section (music + meditation videos)
- 📝 Personal journal for thoughts
- 🎮 Mini relaxation game
- 💡 Motivational quotes

---

## 🖥️ Tech Stack

- Frontend: React (Vite), HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Storage: LocalStorage / Supabase
- UI Design: Glassmorphism + Soft Pastel Theme

## 🚀 Run & Deployment

### 🔹 Run Locally

1. Install dependencies

npm install



2. Start development server

npm run dev


👉 Open in browser: http://localhost:5173/
